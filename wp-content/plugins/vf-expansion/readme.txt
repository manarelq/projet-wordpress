== Description ==

This is a plugin created for Vf themes. This plugin provides additional frontpage sections for Vf Themes.


== Installation ==

= Option 1. =

- Upload the entire `vf-expansion` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Vf Expansion" —> Click install.
- After plugin is downloaded, click active plugin
